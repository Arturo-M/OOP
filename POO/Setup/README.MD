# Pasos para descargar, instalar y correr Visual studio code

## Paso 1: Verifica el sistema operativo de su computador antes de comenzar la descarga e instalación de Visual Studio Code.

## Paso 2: Entrar a la página oficial de Visual studio code

### https://code.visualstudio.com/

![captura1](https://github.com/Arturo-M/OOP/blob/master/POO/Setup/img/Screenshot_1.png)

## Paso 3: Dar click en el botón que dice descargar para Windows.

![captura2](https://github.com/Arturo-M/OOP/blob/master/POO/Setup/img/Screenshot_2.png)

## Si no tiene Windows selecciona la opción de alado que se le indica y descargue según la plataforma de su ordenador.

![Captura3](https://github.com/Arturo-M/OOP/blob/master/POO/Setup/img/Screenshot_3.png)

## Si su ordenador tiene un sistema operativo de 32 bits seleccione la opción de “Otras descargas”, se abrirá otra página en donde vendrán las opciones de descarga, elija la de su conveniencia.

![Captura4](https://github.com/Arturo-M/OOP/blob/master/POO/Setup/img/Screenshot_4.png)

## Una vez seleccionado la opción a descargar se abrirá otra página y su descarga comenzará automáticamente.

![Captura5](https://github.com/Arturo-M/OOP/blob/master/POO/Setup/img/Screenshot_5.png)

## Paso 4: Dar click derecho al archivo descargado y seleccionar donde dice “Mostrar en carpeta”.

![Captura6](https://github.com/Arturo-M/OOP/blob/master/POO/Setup/img/Screenshot_6.png)

## Ahí lo mandara a la carpeta en donde se descargó el archivo de Visual Studio Code.

![Captura7](https://github.com/Arturo-M/OOP/blob/master/POO/Setup/img/Screenshot_7.png)

## Paso 5: Seleccionamos el archivo descargado en la carpeta, damos “ENTER” y nos aparecerá el acuerdo de licencia del Visual Studio Code, leemos la información que se nos proporciona y una vez leído damos al botón donde dice “Acepto el acuerdo” y después damos a “Siguiente”.

![Captura8](https://github.com/Arturo-M/OOP/blob/master/POO/Setup/img/Screenshot_8.png)

## Paso 6: Selecciona “EXAMINAR” para buscar una carpeta donde desee instalara Visual Studio Code o seleccione “SIGUIENTE” para instalar en la carpeta que se le proporciono.

![Captura9](https://github.com/Arturo-M/OOP/blob/master/POO/Setup/img/Screenshot_9.png)

## Paso 7: Seleccione “EXAMINAR” para indicar la carpeta en donde se colocarán los accesos directos del Visual Studio Code, si no desea crear una carpeta, marque el cuadro en donde dice “No crear una carpeta en el Menú Inicio” o si quiere mantener la carpeta en donde se le indica, de click en “SIGUIENTE”.

![Captura10](https://github.com/Arturo-M/OOP/blob/master/POO/Setup/img/Screenshot_10.png)

## Paso 8: Seleccione las opciones que no están marcadas para permitir que Visual Studio Code:
## 1.	Cree un acceso directo en el escritorio.
## 2.	Agrega la opción de “abrir con code” al menú contextual de archivo del explorador de Windows.
## 3.	Agrega la opción de “abrir con code” al menú contextual de directorio del explorador de Windows.
## 4.	Registrar code como editor para archivos que sean admitidos.

![Captura11](https://github.com/Arturo-M/OOP/blob/master/POO/Setup/img/Screenshot_11.png)

## Paso 9: Una vez concluido los pasos para preparar la instalación le aparecerá un marco con todos los preparativos que se hicieron y ya puede comenzar la instalación, de click en el botón e “INSTALAR”, si hay algún cambio que desee realizar antes de instalar solo tiene que oprimir el botón de “ATRÁS”, si no quiere instalar Visual Studio Code seleccione la opción de “CANCELAR”

![Captura12](https://github.com/Arturo-M/OOP/blob/master/POO/Setup/img/Screenshot_12.png)

## Paso 10: Una vez instalada Visual Studio Code crearemos una carpeta en el escritorio del ordenador con el nombre que usted desee.

![Captura13](https://github.com/Arturo-M/OOP/blob/master/POO/Setup/img/Screenshot_13.png)

## Daremos click derecho y seleccioaremos “OPEN WITH CODE”

![Captura14](https://github.com/Arturo-M/OOP/blob/master/POO/Setup/img/Screenshot_14.png)

## Esa carpeta se abrirá con el Visual Studio Code

![Captura15](https://github.com/Arturo-M/OOP/blob/master/POO/Setup/img/Screenshot_15.png)

## Paso 11: Dentro de Visual Studio Code seleccionaremos la opcion “TERMINAL” y abriremos una nueva terminal

![Captura16](https://github.com/Arturo-M/OOP/blob/master/POO/Setup/img/Screenshot_16.png)

## Se nos abrirá una terminal en donde estaremos dentro de la carpeta que usted creo y donde podremos agregar los programas para manejar Visual Studio Code, a continuación, crearemos una carpeta dentro de la carpeta creada utilizando el programa “dotnet new proyect -o (Nombre de la carpeta)” 

## Por ejemplo: mi carpeta que creare se llamara “VSC”, escribo “dotnet new proyect –o VSC”

![Captura17](https://github.com/Arturo-M/OOP/blob/master/POO/Setup/img/Screenshot_17.png)

## Paso 12: Para entrar a la carpeta creada desde Visual Studio Code escribiremos “cd. (Nombre de la carpeta)”

![Captura18](https://github.com/Arturo-M/OOP/blob/master/POO/Setup/img/Screenshot_18.png)

## Paso 13: Para ver el programa daremos click a la carpeta que se creó y se abrirá un menú, seleccionaremos la que dice “Program.cs” y se abrirá el programa que contiene.

![Captura19](https://github.com/Arturo-M/OOP/blob/master/POO/Setup/img/Screenshot_19.png)

## Paso 14: Para verificar que el programa funcione, en la terminal escribiremos “dotnet run” esto activara el programa y debe de aparecer la frase “¡Hello wordl!” (Si desea puede cambiar la frase en la programación)

![Captura20](https://github.com/Arturo-M/OOP/blob/master/POO/Setup/img/Screenshot_20.png)
